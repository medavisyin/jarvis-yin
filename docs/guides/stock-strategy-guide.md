# Jarvis 股票推荐策略指南（小白版）

写给**完全不懂炒股**的你：用最通俗的话讲清楚 Jarvis 的"左侧推荐""右侧推荐"到底是什么、怎么做出来的、你怎么用。技术细节另见 `docs/stock-modules/` 下各模块文档。

> ⚠️ **合规提示**：本文只是解释系统原理与用法，**不构成任何投资建议**。股市有风险，所有推荐都只是参考，买卖决策和盈亏由你自己负责。

---

## 1. 先搞懂两个核心概念：左侧交易 vs 右侧交易

这是理解 Jarvis 两套推荐的钥匙。

### 1.1 用一个生活比喻

想象你要买一件衣服，商店正在打折：

| | 做法 | 比喻 |
|---|---|---|
| **左侧交易** | 衣服还在降价，你**趁便宜先买**，赌它以后会涨回去 | "抄底"——在下跌中买入 |
| **右侧交易** | 衣服已经跌到底、**开始涨价了**，你确认趋势再跟进 | "追涨"——在上涨确认后买入 |

两种思路没有绝对优劣，只是**入场时机**不同：

- **左侧**买得便宜，但要忍受"可能还会继续跌"的煎熬，且不知道底在哪。
- **右侧**买得贵一点，但"涨势已经确认"，心里更踏实，代价是利润少一截、且可能买到假突破。

### 1.2 股市里的对应

| | 左侧交易 | 右侧交易 |
|---|---|---|
| **什么时候买** | 股价下跌/回调时 | 股价已经企稳转涨时 |
| **核心信念** | "便宜就是安全边际" | "趋势是朋友，确认了再上" |
| **关键信号** | 估值低（PE 低）、回调充分、主力悄悄吸筹 | 主力资金由流出转流入、价格站上均线、放量突破 |
| **风险** | 可能抄在半山腰，继续跌 | 可能追在顶部，假突破被套 |
| **Jarvis 对应** | **AI 股票推荐（短期/左侧）** | **AI 股票推荐（右侧交易）** |

> 💡 一句话记忆：**左侧 = 买跌（赌反弹），右侧 = 买涨（跟趋势）**。

### 1.3 为什么 Jarvis 要做两套？

因为**没有任何一种策略在所有行情下都好用**：

- 震荡市、熊市末段 → 左侧抄底更划算（便宜货多）。
- 牛市初段、主升浪 → 右侧追涨更稳（趋势确认）。

Jarvis 让你**一次扫描同时拿到两份报告**，左右互补，你根据当前行情自己判断该偏向哪一边。

## 2. 主力资金：两套策略共同关注的"聪明钱"

### 2.1 什么是主力资金？

股市里有大资金（机构、基金、游资）和小资金（散户）。大资金的买卖会留下痕迹——**主力资金净流入/流出**。Jarvis 通过 `china_market_data` 模块抓取这个数据。

- **主力净流入** = 大资金在买入 → 可能看好后市
- **主力净流出** = 大资金在卖出 → 可能准备撤退

### 2.2 聪明钱的四个阶段

Jarvis 把主力行为分成四个阶段：

| 阶段 | 特征 | 含义 |
|---|---|---|
| **布局期** | 资金悄悄流入，价格还没涨 | 主力在"吸筹"——偷偷低价买货。**左侧最佳买点** |
| **拉升期** | 资金流入，价格已大涨 | 主力开始拉升。追高风险大 |
| **出货期** | 资金流出，价格还在涨 | 主力在"出货"——边拉边卖。**千万别买** |
| **观察期** | 信号不明 | 等待 |

> 💡 这就是为什么 Jarvis 的左侧推荐特别看重"布局期"——在主力还没拉升、价格还便宜时买入。

### 2.3 左侧 vs 右侧怎么看主力资金？

- **左侧推荐**：找"布局期"或"回调+资金悄悄流入"的股票（资金进、价格没涨）。
- **右侧推荐**：找"前期资金流出、**最近 3 天资金转正持续流入**"的股票（资金由出转进，趋势反转确认）。

注意区别：左侧要资金"悄悄进"、价格"还没涨"；右侧要资金"由出转进"、价格"已经企稳转涨"。

## 3. Jarvis 的统一推荐：一次扫描，两份报告

### 3.1 入口在哪？

在 Jarvis 网页的 Stock 工具栏点 **「🤖 AI 推荐」** 按钮，打开统一推荐面板。

### 3.2 它做了什么？

一次扫描内部做了这些事（你不用记，知道大概就行）：

```
① 共享行情抓取：一次拉全 A 股 5000+ 只股票的当日行情（左右侧共用，不重复抓）
        ↓
② 左侧分支                      ③ 右侧分支
   短期(左侧)扫描                  右侧交易扫描
   - 找便宜+回调+吸筹              - 找资金反转+趋势确认
   - DeepSeek 判断                 - DeepSeek 右侧判断
        ↓                              ↓
④ 输出两份报告：
   左栏：左侧·短期推荐（0~5 只）
   右栏：右侧·交易推荐（0~若干只）
```

**关键好处**：行情数据只抓一次（省时间），但两套策略各自独立筛选、独立判断、独立出报告，互不干扰。

### 3.3 怎么操作？

1. 打开「🤖 AI 推荐」面板
2. 勾选 **「💥 DeepSeek」**（推荐勾上，判断质量高很多，成本每次约 1~2 万 tokens，很便宜）
3. 点 **「开始扫描」**
4. 看进度条下方的文字，会实时显示当前阶段：
   - `🌐 共享行情抓取...`
   - `◀ 左侧短期扫描: Layer 2: 详细分析中 (12/100)`
   - `▶ 右侧交易扫描: 资金反转分析...`
   - `完成`
5. 完成后左右两栏分别出报告

> 💡 如果卡在"扫描进行中"很久不动，看进度条下方文字——它会告诉你具体在干嘛。如果文字也不变，可能是网络抓行情慢，等一下；实在不动就点「停止」再重来。

### 3.4 扫描要多久？

- 左侧（短期）：通常 15~30 分钟（要分析 100 只候选 + DeepSeek 判断 + 深度复核）
- 右侧：通常 2~5 分钟（候选少，逻辑轻）
- 共享行情抓取：几十秒到 1 分钟（取决于网络）

## 4. 左侧·短期推荐报告怎么看

每只推荐股票是一张卡片，从上到下：

| 内容 | 含义 | 小白怎么用 |
|---|---|---|
| **#编号 + 名称(代码)** | 推荐排序 | #1 是评分最高的 |
| **综合得分 X/100** | AI 综合打分 | ≥60 才会出现在推荐里；越高越好 |
| **¥价格 + 涨跌幅%** | 当日行情 | 看是否已经涨太多 |
| **PE** | 市盈率（估值） | 越低越"便宜"，但太低可能有问题 |
| **基本面/技术/情绪得分** | 各维度分 | 都高=多维度共振，更靠谱 |
| **🔬 DeepSeek 判断 / 🤖 本地LLM** | 谁做的判断 | DeepSeek 质量更高 |
| **🔄 深度复核: 看多(85%)** | 二次复核结论 | 看空=已被否决不会出现；看多+高置信=更可信 |
| **💡 推荐理由** | AI 给的理由 | 必读！看逻辑是否站得住 |
| **⚠️ 风险** | AI 提示的风险 | 必读！了解可能跌的原因 |
| **📋 操作策略** | 仓位/持有周期建议 | 参考，结合自己风险承受力 |
| **📊 建议买入区间 ¥X~¥Y** | AI 给的买价范围 | 参考，不绝对 |
| **⭐ 自选按钮** | 加入关注列表 | 想持续跟踪就点 |

### 4.1 为什么有时是"暂无推荐"？

**这是正常的，而且是好事。** Jarvis 的设计哲学是"宁可 0 推荐，也不推荐看起来强但不宜买的"。经过三层筛选没有合格标的，就返回 0 —— "不推荐"本身就是最好的建议。

### 4.2 左侧的"硬规矩"（资金面门控）

为了防止 AI 在关键数据缺失时乱推荐，左侧有硬性规矩：

- **资金流向数据缺失** → 不允许出"买入"推荐（降级为"观望"）
- **主力处于"出货期"** → 不允许推荐
- **近 3 日主力大幅净流出** → 不允许推荐
- Top5 推荐**还会再过一道 DeepSeek 深度复核**，如果深度分析看空，就否决

所以你看到的左侧推荐，至少在资金面和深度复核上是被把关过的。

## 5. 右侧·交易推荐报告怎么看

右侧卡片字段类似，但重点不同：

| 内容                         | 含义          | 小白怎么用                   |
| -------------------------- | ----------- | ----------------------- |
| **综合得分**                   | 右侧复合打分      | 资金反转占 50% 权重，所以高分≈反转强   |
| **资金反转: 3日净流入 X亿 / 占比 Y%** | **核心信号**    | 近 3 天主力由出转进，占比≥3% 才入选   |
| **站上MA5**                  | 价格在 5 日均线之上 | 短期趋势转强的标志               |
| **MA20: ¥X**               | 20 日均线位置    | 价格在 MA20 附近=待突破；站上=突破确认 |
| **🛑 严格止损 ¥X**             | **必须执行**    | 右侧交易铁律：跌破止损马上走          |
| **🎯 2-3个月目标 ¥X**          | AI 给的目标价    | 参考，到了要主动止盈              |
| **🔬 DeepSeek 右侧判断**       | 谁做的判断       | DeepSeek 质量更高           |

### 5.1 右侧的入场条件（必须同时满足）

1. **前期 10 天主力资金是净流出**（说明之前在跌/被卖）
2. **最近 3 天主力转为持续净流入**（资金开始回来）
3. **3 日主力净占比 ≥ +3%**（回流力度够强，不是蜻蜓点水）
4. **价格站上 MA5**（短期趋势确认）
5. （加分项）逼近/突破 MA20、放量

> 💡 翻译成大白话：**"之前被抛售的股票，最近几天大资金开始持续回流，而且价格已经止跌转涨"** —— 这就是右侧入场的信号。

### 5.2 右侧的交易纪律（非常重要）

右侧交易**最忌讳死扛**：

- **严格止损**：跌破止损价立刻走，不要"等回本"。
- **目标止盈**：到了目标价主动卖，别贪。
- **持有周期**：通常 2 周 ~ 2-3 个月，不是长线。
- **无信号不入场**：如果今天扫描结果是"暂无右侧推荐"，说明今天没有资金反转标的——**耐心等**，不要硬找。

### 5.3 为什么右侧经常是"暂无推荐"？

因为右侧条件**很严格**——要"前期流出+近期转正+力度够+趋势确认"同时成立。大部分股票大部分时间不满足。这是**正常的**，右侧交易讲究耐心等待确认信号，无信号即不入场。

## 6. 左右两侧怎么配合用？

```
你打开「AI 推荐」→ 开始扫描
        ↓
   拿到两份报告
        ↓
┌───────────────┬───────────────┐
│ 左侧有推荐     │ 右侧有推荐     │
├───────────────┼───────────────┤
│ 偏向"抄底"     │ 偏向"跟趋势"   │
│ 适合震荡/回调市 │ 适合主升浪/牛市初│
└───────────────┴───────────────┘
        ↓
  你的判断：
  - 当前大盘在跌/震荡 → 优先看左侧
  - 当前大盘在涨/启动 → 优先看右侧
  - 两边都有 → 说明市场机会多，可分散配置
  - 两边都无 → 说明今天没好机会，观望
        ↓
  选定后：读理由+风险 → 设止损+仓位 → 分批买 → 严格执行纪律
```

### 6.1 一个重要提醒：一致性

Jarvis 的左侧 Top5 会经过深度分析复核，**保证不会出现"短期推荐买入但深度分析看空"的矛盾**。如果你在某只股票上看到左右侧结论不一致，那是正常的——它们本来就是**不同时机**的策略，不是对同一只股票的"同一个判断"。

## 7. 小白常见问题

**Q：我应该信左侧还是右侧？**
A：看你对自己的判断。能忍受浮亏、会判断估值 → 左侧；想要趋势确认、执行纪律强 → 右侧。新手建议**右侧为主**（信号更明确），左侧小仓位试。

**Q：推荐了一定会涨吗？**
A：不会。任何策略都是概率，不是确定性。务必设止损、控仓位。

**Q：为什么有时扫描很久？**
A：左侧要分析 100 只候选 + 多轮 DeepSeek 调用，慢是正常的。看进度条下方文字确认在推进就行。

**Q："暂无推荐"是不是系统坏了？**
A：不是。Jarvis 宁可不出推荐也不乱推。0 推荐是正常的、负责任的结果。

**Q：DeepSeek 要勾吗？**
A：强烈建议勾上。判断质量明显更高，成本每次才 1~2 万 tokens，很便宜。前提是你在齿轮⚙设置里配了 DeepSeek API Key。

**Q：报告里的"买入区间"准吗？**
A：是 AI 根据数据给的参考，不是绝对。可以当锚点，但最终买价结合你自己看盘。

**Q：我想加自己的策略怎么办？**
A：Jarvis 预留了策略插件接口，见开发指南 `docs/guides/stock-new-strategy-guide.md`。

## 8. 风险提示（必读）

- **所有推荐都是参考，不是投资建议**。Jarvis 是辅助决策工具，不是自动赚钱机器。
- **股市有风险，过往表现不预示未来**。模型在历史数据上表现好，不代表未来一定好。
- **务必设止损、控仓位**。单只股票别押太重，首次建仓分批。
- **多维度交叉验证**。不要只看一个信号就下手，结合大盘、板块、消息面。
- **执行纪律比预测更重要**。止损到了就走，别抱"等回本"幻想。
